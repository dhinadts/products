# Balance Sheet Ledger System

A Balance Sheet based ledger platform for a small-scale industry, built with Indian accounting terminology and strong security, compliance, and audit capabilities.

## Contents
- `docs/` - Architecture, low-level design, detailed design documents.
- `frontend/` - Flutter frontend scaffold.
- `backend/` - Service-oriented backend scaffold.

## Goals
- Asset/liability ledger with Debit/Credit reporting.
- JWT-based authentication with RBAC.
- MongoDB schema validation and AES-256 encryption at rest.
- Gmail App Password email alerts and Firebase notifications.
- Docker-based deployment with AWS infrastructure.
